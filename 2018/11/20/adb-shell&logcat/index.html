输入

手机输入框 focus 时
$ adb shell input text “hello,world”
输入 hello
注：空格、’、&都是特殊字符，需要用 keyevent 输入。且只能输入 ascii 码的字符
$ adb shell input keyevent 61
tab 跳到下一个输入框
$ adb shell input keyevent KEYCODE_SPACE
输入空格
