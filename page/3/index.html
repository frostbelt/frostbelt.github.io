发表于 2019-06-11 | 热度 ℃
| 字数统计 80 | 阅读时长 1

做一个 sticky 效果，在 Android 小程序中出现页面滚动时渲染延迟

原因：小程序在 Android 下，滚动会导致所有异步阻塞，包括 setInterval、setTimeout 等。

解决：在页面配置文件里（即.json文件）添加 usingComponents 字段即可解决问题。

win10 解决 Failed to register Fiddler as the system proxy.

发表于 2019-05-16 | 热度 ℃
| 字数统计 96 | 阅读时长 1

win10 安装 fiddler 后提示 “Failed to register Fiddler as the system proxy.”

需要在服务中启动 WinHTTP Web Proxy Auto-Discovery Service 服务
如果不能启动该服务，可能是被 360安全卫士 干掉了
按以下操作解除禁用:
360安全卫士 -> 优化加速 -> 启动项 -> 系统关键服务 -> 找到该服务并恢复启动 -> 重启
